fn-map-test

(fn-map-test fn-map & data-maps)

For each key in fn-map, apply the function value to the corresponding entries in the data maps. Return new data map with boolean values.

near-same

(near-same myfn margin val1 val2)(near-same myfn val1 val2)

True if the two values are within the given margin. If the function argument is given, true if the values are within the margin when the function is applied to both values.

(simple-vtext-block-recognizer vtexts)
