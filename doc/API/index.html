DRAY 0.5.0-SNAPSHOT

DRAY (Diagrammatic Representation and Analysis) is a framework for performing graphical and structural analysis of PDF and other document files. It is currently in development.

Charts to display useful textual characteristics of a given set of visual elements. Can also be used to convert visual elements, working sets, or PDFs into Incanter data sets.
