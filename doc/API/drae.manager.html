drae.manager

dm-add-overlay

(dm-add-overlay dm overlay-map)

Overlay a top-level working set overlay onto the working set in this data manager.

dm-restore-ws-from-overlay

(dm-restore-ws-from-overlay dm json-file)(dm-restore-ws-from-overlay dm)

Restore the set of working sets described in the JSON file. Returns the modified data manager object. If the second argument is omitted, load the default ws-overlay file if it exists.

dm-save-ws-to-overlay

(dm-save-ws-to-overlay dm json-file)

Save given data manager’s current working sets as an overlay in a JSON file. Json-file can be either a file object or string. Returns the data manager.

glean-visual-elements

(glean-visual-elements rep)

Return the Visual Elements in the given structured representation.

json->overlay

multimethod

Read in either a JSON string or a file as a new working set overlay.

make-data-manager

multimethod

Return a new data manager given either a pdf file, a filename, or a VDocument instance.

make-layer

(make-layer name representation elements)(make-layer name representation)(make-layer name)

Create a new layer object with a given name, representation, and element set. Omitted arguments default to empty lists. Keywords in the representation map are converted to strings.
