dray.wset

ws->json

(ws->json ws)

Write working set as re-readable JSON string. Returns a working set for the second argument, which must be a document-level working set.

ws->map

(ws->map ws)

Write working set as a re-readable map structure.

ws-children

(ws-children ws & {:keys [tag], :or {tag nil}})

All the child working sets of this one. If the :tag keyword is used, only returns those descendants with the given tag.

ws-descendants

(ws-descendants ws & {:keys [tag], :or {tag nil}})

All the descendant working sets of this one. If the :tag keyword is used, only returns those descendants with the given tag.

ws-images

(ws-images ws)

Return all VImages in this working set.

ws-items

(ws-items ws)

Items in the working set.

ws-leaf?

(ws-leaf? ws)

Is this a working set leaf?

ws-make-child

(ws-make-child ws name bbox & {:keys [tags], :or {tags (quote [])}})

Create a child instance of this working set with the given name and bounding box.

ws-page-root

(ws-page-root ws page-no)

The working set for the given page. The ws argument can be any working set in the tree.

ws-page?

(ws-page? ws)

Is this a page-level working set?

(ws-parent ws)

Parent working set for this one (if any).

ws-replace-child

(ws-replace-child parent-ws old-child-ws new-child-ws)

Replace the old child working set with the new one in the parent working set’s children.
