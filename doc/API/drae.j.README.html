drae.j.README

+readme+

This package exists solely to include documentation of the compiled Java classes within drae.j.*. Because metadata is removed from compiled namespaces, it is not possible to include namespace-level documentation for compiled namespaces, so we include it here for the compiled classes within the package.

  • drae.j.Corpus: Class for accessing default corpus libraries.

  • drae.j.Doc: Class for creating VDocument (and associated pages and visual elements).

  • drae.j.Paxll: Class for performing queries to Pathway Commons 2 server

  • drae.j.BoundingBox: A bounding box (given as a field in most visual elements). Subclass of java.awt.geom.Rectangle2D.Double.

Within the drae.j.Doc namespace, there are also a set of classes for documents and their parts.

All visual elements are instances of the drae.j.VisualElement.El interface, which return proper values for the getBbox and getItems methods:

  • drae.j.VisualElement.VDocument: A single document. Getters: filename, getItems

  • drae.j.VisualElement.VPage: A single page within a document. Getters: getBbox, getItems

  • drae.j.VisualElement.VText: A single line of text within a page. Getters: getBbox, getItems, text

  • drae.j.VisualElement.VTextToken: A single word token within a text line. Getters: getBbox, text

  • drae.j.VisualElement.VDiagram: ( Not currently implemented ) A diagram from the paper (image with associated text lines). Getters: getBbox, image, getItems

  • drae.j.VisualElement.VImage: A single image from the page. Getters: bitmap-path getBbox. To get the complete image path from a VImage, call full-bitmap-path-for on the image and it’s containing document.

  • drae.j.VisualElement.Blob: ( Not currently implemented ) A single blob detected in an image.

A bounding box is a subclass of Rectangle.Double2D, and so can be accessed using those methods, including getX, getY, getWidth, getHeight.

The easiest way to create a VDocument is to call the drae.j.Doc/getVDocument method on a pdf file instance, such as that returned by drae.j.Corpus/getCorpusFile method.
