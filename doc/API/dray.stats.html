dray.stats

grouping-selectors

All grouping selector keywords.

groupings

(groupings selector vels)

Return a map-based table of visual elements by selector. Selector can either be a function that takes a visual element and returns a value, or a keyword for grouping-selector that will return a selector function from that table.

line-spacing-frequencies

(line-spacing-frequencies vels)

Frequencies for difference in Y for sequential elements. This is broken out separately because it involves distances between visual elements (e.g., text rows) rather than single-item dimensions. May not be working correctly.

print-table

(print-table m)

Print out frequency or other map as formatted table.

proportions

(proportions selector vels & _)

Return an ordered map of value proportions for the given selector.

prune-by-proportion

(prune-by-proportion table cutoff)(prune-by-proportion table)

Remove entries that contain less than the cutoff coverage (default: 0.1 = 10% cutoff). Argument is groupings map (will also use frequency and proportion data if already in table).
