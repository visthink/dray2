drae.ssa

Simple spatial analysis module. Uses the prxml data structures for the diagram labels and blobs to detect some simple spatial relationships.

adjoining-labels

(adjoining-labels label-data)

Given the PRXML data that contains a label (or set of labels) find the set of labels that are touching or close to touching.

filter-value-range

(filter-value-range seq key-fn v-low v-high)

For the given sequence, return the sequence of all items that contain the given key value within the given range. Does not assume an ordered sequence, so expense is linear with size of list.
