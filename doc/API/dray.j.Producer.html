make-producer

(make-producer key name doc fn)(make-producer name doc fn)

Create a single producer entry object. Key should be either a keyword, string, or symbol. If the key is omitted, uses the name as the key (this is for backward compatibility. Producer entries can be accessed via the .key, .name, .doc, and .fn methods.
