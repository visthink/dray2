drae.gui

Routines for accessing DRAE GUI elements from Clojure.The main GUI routines are in Java portion of the code, but these routines make it easy for Clojure code to pull out key facts from the current GUI. Assumes a single GUI instance.

This package also includes the routines for several default DRAE toys.
